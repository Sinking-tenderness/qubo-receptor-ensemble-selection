# Stage 2 Receptor Preparation Record: CDK2 1HCL

## Purpose

This module prepares aligned CDK2 1HCL chain A as a rigid AutoDock Vina
receptor under the same preparation policy used for the validated 1AQ1
baseline.

The goal is protocol consistency. This preparation does not establish that
1HCL has better docking scores or virtual-screening enrichment than 1AQ1.

## Input

- Input: `receptors/prepared/1HCL_A_aligned_to_1AQ1.pdb`
- Input SHA-256: `D60C8A340167A35D06FA9DD6059BE062AAA7DF54056FF8BCAA373C2E328A840F`
- Protein chain: A
- Crystallographic waters before cleaning: 180
- Non-water HETATM records: 0
- Modeled protein residues: 294
- Missing residues: A:37-40

The aligned input is generated by `scripts/align_receptor_structure.py` from
the RCSB 1HCL coordinates documented in `cdk2_1hcl_alignment_record.md`.

## Alternate-Location Decision

Two residues contain alternate atom locations:

| Residue | Location A occupancy | Location B occupancy | Pocket relevance |
| --- | ---: | ---: | --- |
| GLN131 | 0.80 | 0.20 | In the ATP pocket |
| SER264 | 0.66 | 0.37 | Outside the ATP pocket |

The controlled baseline selects location A for both residues:

```text
A:131=A,A:264=A
```

Location B is not labeled incorrect. It is a lower-occupancy experimental
microstate that may be evaluated as a separate receptor conformer later.
Combining A and B coordinates in one rigid receptor would create overlapping
atoms and an invalid docking input.

## Software and Parameters

- Python: 3.11.15
- ProDy: 2.4.1
- Meeko: 0.7.1
- Charge model: Gasteiger
- Receptor mode: rigid
- Flexible residues: none
- ProDy altloc selection: A
- Meeko explicit altloc selection: `A:131=A,A:264=A`
- Water policy: remove all waters for the first controlled two-receptor baseline

The default Meeko residue templates produced the same HIE-like histidine state
for every histidine shared by 1AQ1 and 1HCL. This is a protocol-consistency
observation, not experimental validation of protonation states at a specific
pH.

## Reproducible Command

```powershell
conda run -n qubo-receptor-ensemble python .\scripts\prepare_receptor.py `
  --input-pdb .\receptors\prepared\1HCL_A_aligned_to_1AQ1.pdb `
  --chain A `
  --prody-altloc A `
  --wanted-altloc A:131=A,A:264=A `
  --protein-only-output .\receptors\prepared\1HCL_A_aligned_protein_only.pdb `
  --prepared-pdb-output .\receptors\prepared\1HCL_A_receptor_prepared.pdb `
  --pdbqt-output .\receptors\prepared\1HCL_A_receptor.pdbqt `
  --summary-output .\data\processed\1HCL_A_receptor_preparation_summary.json `
  --charge-model gasteiger `
  --overwrite
```

The script invokes Meeko as `python -m meeko.cli.mk_prepare_receptor` through
the active environment. This avoids relying on a machine-specific launcher
script and verifies that successful return codes actually produced every
required output file.

## Recorded Failures

1. Calling the Windows-generated `mk_prepare_receptor.py` launcher returned a
   zero exit code in this environment but produced no receptor files. The
   validated pipeline therefore calls the Meeko Python module directly and
   checks output existence.
2. Running Meeko without an explicit altloc selection failed because residues
   A:131 and A:264 retained alternate-location annotations. The final command
   records the chosen A locations explicitly.
3. `--allow_bad_res` was not used because it could remove residues that fail
   template matching, including the pocket residue GLN131.

Failures were fixed by making the protocol explicit, not by deleting samples
or suppressing validation.

## Relevance to Multi-Conformer Screening

1AQ1 and 1HCL are now in one coordinate frame and use the same box, water
policy, charge model, atom typing, and rigid-receptor assumptions. The next
module can dock the same 60-ligand library against 1HCL and append a second
receptor column to the ligand-by-conformer score matrix.

Any performance difference must still be interpreted with uncertainty. A
consistent preparation protocol reduces confounding but does not eliminate
scoring-function error or prove that one receptor conformation is biologically
superior.
